Module 01

SQL Injection & DB 보안

동적 문자열 결합 쿼리의 위험성을 체험하고, Prepared Statement(파라미터 바인딩) 기술로 DB 무단 접근 및 쿼리 변조 공격을 완벽히 방어합니다.

6 Steps • SQL Visualizer 실습 시작
Module 02

XSS & 클라이언트 보안

Reflected 및 Event XSS 스크립트 주입 원리를 이해하고, 서버측 HTML Entity Escape (Sanitization) 및 DOM 보안 기법을 적용합니다.

5 Steps • DOM Inspector 실습 시작
Module 03

CSRF & 세션 요청 위조

브라우저 자동 쿠키 전송 특성을 악용한 타 사이트의 위조 요청을 분석하고, Anti-CSRF 검증 토큰 및 SameSite 쿠키 정책을 실습합니다.

5 Steps • HTTP Inspector 실습 시작
Module 04

인증/인가 & IDOR 방어

식별자 파라미터 조작(IDOR)을 통한 타인 데이터 무단 접근 사례와, 백엔드 세션 인가(RBAC) 검증 구현 기법을 통찰합니다.

5 Steps • API Inspector 실습 시작
