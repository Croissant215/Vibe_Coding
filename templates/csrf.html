-- 왼쪽에서 송금 요청을 전송하면 백엔드로 전송되는 HTTP 패킷과 쿠키, CSRF 토큰이 관찰됩니다.
@app.route('/api/vulnerable-transfer')
def vulnerable_transfer():
    # ❌ 토큰 검증 없이 자동 포함된 세션 쿠키만으로 송금 승인
    return jsonify({'success': True, 'mode': 'vulnerable'})
@app.route('/api/secure-transfer')
def secure_transfer():
    csrf_token = request.headers.get('X-CSRF-Token')
    if csrf_token != 'VALID_TOKEN':
        # ✅ 외부 타 사이트의 위조 요청은 토큰이 없으므로 거절
        return jsonify({'success': False, 'message': 'Invalid CSRF Token'}), 403
